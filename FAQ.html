Frequently Asked Questions

Packages

What is "Feature Support"?


Feature Support is a tier'd system that allows the easy categorization of packages based on how many features they have. The following tier's are available:


  • Basic
  • Advanced
  • Advanced Pro

    • If a package includes Basic Feature Support, it includes only basic features such as app icons, settings icons, and app badges. These packages have the least impact on battery and performance, while requiring the least amount of storage.


    If a package includes Advanced Feature Support, it includes some advanced features like keyboard sounds, boot logos, and docks. These packages can have moderate impact on battery and performance, while requiring an average amount of storage.


    If a package is labeled as Advanced Pro, it includes the entire array of advanced features. These packages typically have a higher impact on battery and performance, and can require a large amount of storage.


    What is "Super Resolution"?


    Packages that support Super Resolution offer the highest resolution possible, typically equivalent to native quality.

    Pricing

    How much does HexaCore charge for packages?


    At HexaCore, we believe that everyone should have access to high quality, feature-filled packages. This is why we will never charge a fee of any kind. If you paid for a package that belongs to HexaCore, please report it here.

    Common Issues

    Safari says "address invalid" when trying to add repo.


    If you are experiencing this error, check to make sure that your device is in a jailbroken state. Most times, this error is a result of someone trying to add a repo to an unjailbroken device.


    This error can also be triggered by attemtping to add a repo using Safari on Mac.


    My package manager says "DPKG LOCKED."


    This error can be fixed by simply refreshing your sources.


    I have an issue, how do I report it?


    If you have an issue, the best way to report it is to reach out via email or Twitter. You should expect a response within 48 hours, but some may take longer.
