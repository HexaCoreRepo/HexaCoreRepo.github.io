Big Sur in Your Pocket.


Mobile Sur contains over 30 icons taken directly from macOS Big Sur to ensure the sharpest resolution possible as well as beautiful vibrancy. Mobile Sur also includes official Big Sur system sounds for charging and locking your device, the full set of macOS Big Sur Settings icons, and the macOS Big Sur dock* for iPhone’s and iPod Touch’s with Touch ID.


Mobile Sur also provides unmatched support. Mobile Sur is updated consistently ensuring the best possible user experience, while regularly delivering new features. You can even request icons to be added - as long as they are currently present on macOS Big Sur.
